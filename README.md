# Детективное агентство «Тайна»

Учебный проект на **Bottle**
## Содержимое
- `index.html` - Главная
- `about.html` - О нас  
- `contact.html` - Контакты
- `team.html` - Сотрудники
- `site.css` - Стили

## Установка
1. Скачать архив
2. Открыть `index.html` в браузере

- HTML5
- Bottle
- Visual Studio
- Github
- CSS3

Сайт работает локально без сервера.
