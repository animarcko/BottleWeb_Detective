# Детективное агентство «Тайна»

Корпоративный **многостраничный веб-ресурс**, предоставляющий полную информацию об организации в удобном и структурированном формате. Проект создан для отработки работы с **системой контроля версий Git**.

## Описание проекта
- **Тип сайта**: Корпоративный многостраничный
- **Технологии**: HTML5, CSS3, Bottle (локальная разработка)
- **Среда разработки**: Visual Studio
- **Разработчики**: Маргарита Скулябина, Андреева Екатерина группа с324

## Установка и запуск
1. Клонируйте репозиторий: `git clone https://github.com/animarcko/repo.git`
2. Распакуйте файлы в любую директорию
3. Откройте **`index.html`** в браузере (Chrome, Firefox, Edge)

## Инструкции по доработке

**Добавление сотрудника** (`team.html`):
Для добавления карточки нового сотрудника скопируйте следующий блок в раздел с командой на странице `team.html`. Замените фото, имя, должность и описание на актуальные данные.
```html
<div class="team-card">
    <img src="images/photo.jpg" alt="Фото сотрудника">
    <h3>Имя Фамилия</h3>
    <p class="position">Должность</p>
    <p class="description">Описание опыта.</p>
</div>
```
**Изменение цветов** (`site.css`):
Все стили централизованно управляются через файл site.css. Для изменения цветовой схемы отредактируйте переменные в начале файла:
```css
:root {
    --primary-color: #1a2a3a;
    --accent-color: #c9a03d;
    --text-light: #f5f5f5;
}
```
