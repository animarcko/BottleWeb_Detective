# Детективное агентство «Тайна»

Проект создан в качестве учебного задания для отработки работы с системой контроля версий Git. Тематика детективного агентства выбрана в соответствии с одним из предыдущих заданий по учебной практике. Сайт представляет собой корпоративный многостраничный веб-ресурс, предоставляющий полную информацию об организации в удобном и структурированном формате.

## Содержимое
- `index.html` - Главная
- `about.html` - О нас  
- `contact.html` - Контакты
- `team.html` - Сотрудники
- `site.css` - Стили

## Установка
Сайт является полностью статическим и не требует настройки серверного окружения.
1. Скачайте архив с проектом или выполните клонирование репозитория
2. Распакуйте файлы в любую директорию на вашем компьютере
3. Откройте файл index.html в любом современном веб-браузере (Google Chrome, Mozilla Firefox, Safari, Edge)

Для корректного отображения всех страниц рекомендуется открывать сайт именно через файл index.html, а не отдельные HTML-документы.

## Используемые технологии
- HTML5 — семантическая разметка страниц
- CSS3 — стилизация, адаптивная верстка, визуальное оформление
- Bottle — использовался в качестве легковесного инструмента для локальной разработки
- Visual Studio — среда разработки
- Git / GitHub — система контроля версий и хостинг репозитория


## Развертывание
Проект поддерживает размещение на следующих платформах:
- GitHub Pages — автоматическое развертывание при пуше в репозиторий
- Netlify — поддерживается развертывание через Drag & Drop или подключение к репозиторию

Для публикации достаточно загрузить содержимое проекта в корневую директорию хостинга статических сайтов.

## Особенности проекта
- Единая стилизация всех страниц через внешний файл site.css
- Отсутствие зависимостей от внешних библиотек и фреймворков
- Кросс-браузерная совместимость
- Структура, готовая к дальнейшему расширению функционала
